# RAG GenAI Chatbot

## Description
The chatbot is a virtual assistant that can answer the user queries from the uploaded pdf document.
We can store and retrieve the responses from the documents using this Retrieval Augmented Generation based Chatbot.

## Getting Started

### Setup Instructions

1. Virtual environment creation - python -m venv venv
2. Activate virtual environment - venv\Scripts\activate
3. Install dependencies - pip install -r requirements.txt
4. Run application - python app.py
